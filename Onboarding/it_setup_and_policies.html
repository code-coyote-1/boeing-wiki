Boeing IT Setup and Access Guide

Welcome to Boeing! This guide will help you set up your IT environment and access essential tools.

IT Setup and Access

This guide provides step-by-step instructions for setting up your IT environment at Boeing. It covers everything from hardware setup to software installation and network access. Follow each section carefully to ensure you have everything you need to start working efficiently.

1. Hardware Setup

To begin, you'll need to set up your hardware. Follow these steps:

  • Unpack Your Equipment: Unpack your computer, monitor, keyboard, and other peripherals.
  • Connect to Power: Ensure all devices are plugged into a power source.
  • Assemble Workstation: Connect your monitor, keyboard, mouse, and other peripherals to your computer.
  • Adjust Settings: Configure monitor settings for optimal display.

If you encounter any issues, contact IT Support for assistance.

2. Software Installation

Install the necessary software to perform your job efficiently:

  • Operating System: Ensure your computer is running the latest version of Windows or macOS.
  • Email Client: Install Microsoft Outlook to access your Boeing email account.
  • Productivity Suite: Install Microsoft Office for word processing, spreadsheets, and presentations.
  • Collaboration Tools: Install Microsoft Teams for communication and collaboration with your team.
  • Project Management Software: Access Jira for project tracking and task management.

Visit the Software Downloads page for installation links and instructions.

3. Network Access

Connect to Boeeing's network to access internal resources and services:

  • Wired Connection: Use an Ethernet cable to connect your computer to the network.
  • Wi-Fi Connection: Connect to the secure Wi-Fi network using your Boeeing credentials. SSID: BoeeingSecure
  • VPN Access: Install and configure the VPN client for remote access to the Boeeing network. Follow the VPN Setup Guide for detailed instructions.
  • Accessing Shared Drives: Map network drives to access shared files and resources. Instructions can be found in the Network Drive Mapping Guide.

4. Security Protocols

Security is a top priority at Boeing. Follow these protocols to protect company data:

  • Passwords: Use strong passwords for all accounts and change them regularly. Refer to the Password Policy for guidelines.
  • Two-Factor Authentication (2FA): Enable 2FA for additional security on all compatible services.
  • Software Updates: Regularly update all software and operating systems to the latest versions.
  • Data Encryption: Ensure sensitive data is encrypted when stored and transmitted.
  • Phishing Awareness: Be cautious of phishing emails and report suspicious activity to Security.

5. Troubleshooting Common Issues

Here are solutions to some common IT problems you might encounter:

  • Internet Connectivity Issues: Restart your router or check with IT Support.
  • Software Installation Problems: Ensure you have administrative rights or contact IT Support.
  • Hardware Malfunctions: Check connections, and if unresolved, request a replacement via the IT Support Portal.
  • Email Configuration: Follow the Email Setup Guide if you experience issues accessing your email.

For additional help, refer to the IT Troubleshooting Guide or contact IT Support.
